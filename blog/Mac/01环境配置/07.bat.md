# bat
>类似 cat 但有语法高亮。

## 安装
  在终端中输入：`brew install bat`。
  
  通过别名，用 bat 替换 cat，在 `~/.zshrc` 文件中增加 `alias cat="bat"`（可选）。
  
  选择主题，在 `~/.zshrc` 文件中增加 `export BAT_THEME="Dracula"`。
  
  重新加载 `~/.zshrc`文件，使配置生效：`source ~/.zshrc`。
